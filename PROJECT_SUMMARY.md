# 厂务自动化 - 项目完整文档

## 项目概述

**厂务自动化** 是一个专业的 B2B 企业协作平台，连接制造企业、自动化企业、供应商和人力资源机构。平台提供需求发布、企业资源浏览、以及完整的后台管理系统。

**技术栈:**
- Next.js 16（App Router）
- React 19
- TypeScript
- Tailwind CSS
- Lucide Icons

---

## 功能架构

### 1. 公开网站（Public Pages）

#### 首页 (`/`)
- **英雄区段**: 吸引人的标题和行动按钮
- **企业分类**: 5个主要企业分类卡片（甲方企业、自动化企业、供应商、人力资源、配套服务）
- **平台优势**: 展示3个核心优势特性
- **最新需求**: 展示最新3条需求
- **推荐企业**: 展示3家推荐企业
- **行动号召**: CTA 区段鼓励用户加入
- **联系方式**: 企业微信、热线电话、在线留言
- **页脚导航**: 完整的页脚链接和公司信息

#### 需求列表页 (`/requirements`)
**搜索与过滤:**
- 关键词搜索（标题、企业名称）
- 按类型过滤（全部、招聘、采购、合作、项目、人才）
- 实时搜索结果计数
- 完整的分页功能

**需求卡片:**
- 需求类型标签（彩色）
- 状态标签（进行中、已完成）
- 发布企业名称
- 需求摘要（限制行数）
- 发布日期
- 查看详情按钮

#### 需求详情页 (`/requirements/[id]`)
**内容展示:**
- 需求标题、类型、状态、发布日期
- 完整需求描述（保留格式）
- 企业基本信息

**交互功能:**
- **提交合作需求**: 打开 Dialog 表单
- **联系平台客服**: 按钮链接
- **相关推荐**: 显示2条相关需求

**表单功能:**
- 姓名、电话、公司、合作内容输入
- 表单验证和错误提示
- 提交状态反馈
- 成功确认消息

---

### 2. 后台管理系统 (`/admin`)

#### Dashboard (`/admin`)
**数据概览:**
- 统计卡片（需求总数、企业数、用户数、活跃度）
- 最新需求表格（前5条）
- 最新留言表格（前5条）
- 快速操作按钮

#### 需求管理 (`/admin/requirements`)
**功能:**
- 完整的表格列表显示
- 搜索和筛选功能
- 分页导航

**数据列:**
- 需求标题
- 企业名称
- 需求类型（彩色标签）
- 发布时间
- 状态（待审核/进行中/已完成 - 彩色编码）
- 操作按钮（编辑/删除）

**新增/编辑对话框 (Dialog):**
- **必填字段验证:**
  - 需求标题
  - 企业名称
  - 需求内容
- **可选字段:**
  - 需求类型（下拉选择）
  - 状态（下拉选择）
- **表单特性:**
  - 实时验证提示
  - 错误消息显示
  - 提交加载状态
  - 成功确认动画
  - 关闭按钮（X 图标）

**删除确认对话框:**
- 确认消息
- 取消和删除按钮
- 删除加载状态

#### 留言管理 (`/admin/messages`)
**功能:**
- 完整的表格列表显示
- 搜索功能
- 分页导航

**数据列:**
- 用户姓名
- 电话（脱敏显示）
- 公司名称
- 留言内容（单行显示）
- 来源页面
- 提交时间
- 联系状态（未联系/已联系 - 彩色标签）
- 操作按钮（查看/删除）

**留言详情抽屉 (Drawer):**
- 从右侧滑出动画
- 完整留言信息展示
- 标记已联系按钮
  - 仅在"未联系"状态时显示
  - 标记后显示成功确认
  - 自动关闭抽屉
- 关闭按钮

**删除确认对话框:**
- 与需求管理相同的实现

#### 系统设置 (`/admin/settings`)
- 平台基础配置（未来功能）

## UI 组件和设计模式

### 对话框 (Dialog)
**使用场景:**
- 新增需求
- 编辑需求
- 确认删除
- 提交合作需求

**特性:**
- 黑色半透明背景覆盖
- 居中显示
- 最大宽度限制
- 自动滚动处理长内容
- X 关闭按钮
- 表单验证和错误提示

### 抽屉 (Drawer)
**使用场景:**
- 查看留言详情

**特性:**
- 从右侧滑入
- 全高显示
- 黑色半透明背景
- 自动滚动处理
- X 关闭按钮

### 按钮样式
- **Primary**: 主操作按钮（新增、提交、更新、标记）
- **Secondary**: 次要操作
- **Outline**: 取消和副操作
- **Destructive**: 删除操作（红色文本）

### 表单验证
- 必填字段标记（红色 *）
- 实时错误提示（红色文本）
- 字段聚焦时呈现蓝色边框
- 成功状态视觉反馈

### 加载和成功状态
- **加载**: 按钮显示"...中"文本，禁用状态
- **成功**: 绿色对勾图标 + 确认消息
- **自动关闭**: 延时后自动返回列表

## 数据结构

### 需求对象
```typescript
{
  id: string
  title: string              // 需求标题
  company: string            // 企业名称
  type: string              // 招聘|采购|合作|项目|人才
  status: string            // 待审核|进行中|已完成
  content: string           // 需求描述
  date: string              // 发布日期
  publishedAt: string       // ISO 日期格式
}
```

### 留言对象
```typescript
{
  id: number
  name: string              // 用户姓名
  phone: string             // 电话号码（脱敏）
  company: string           // 公司名称
  content: string           // 留言内容
  source: string            // 来源页面
  submittedAt: string       // 提交日期
  status: string            // 未联系|已联系
}
```

## 快速开始

### 安装依赖
```bash
pnpm install
```

### 启动开发服务器
```bash
pnpm dev
```

### 访问应用
- 公开网站: http://localhost:3000
- 后台管理: http://localhost:3000/admin

### 构建生产版本
```bash
pnpm build
pnpm start
```
